## Recommended policy for project images

1. Prefer `CC0` or `Public Domain` when available.
2. Accept `CC BY` or `CC BY-SA` only if attribution is documented in the repo.
3. Avoid blog, shopping mall, brand reseller, and generic web-search images.
4. For any existing local image, record the original source before shipping or publishing.
